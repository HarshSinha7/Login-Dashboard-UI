What it does
- Handles login, signup, and access to a dashboard securely.
- Validates user input like emails, passwords, and required fields.
- Protects against CSRF attacks with secure tokens in forms.
- Implements safe logout using POST requests to properly end sessions.

Tech Stack
Java 21, Spring Boot, Spring Security 6
Thymeleaf for server-side templates
Lombok for boilerplate-free code
Maven for builds
